# Capstone Project

## Overview

This project is a Flutter application developed as a capstone project and containerized using Docker for consistent deployment across environments.

## Features

* Cross-platform Flutter application
  
* Responsive user interface
  
* Form validation
  
* State management
  
* Local data persistence
  
* Dockerized deployment

---

## Technologies Used

### Frontend

* Flutter
  
* Dart

### Containerization

* Docker
  
* Docker Compose

## Prerequisites

Install:

* Flutter SDK
  
* Docker
  
* Docker Compose
  
* Git

---

## Clone Repository

```bash
git clone <repository-url>
cd project
```

---

## Run Locally

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```

---

## Build Docker Image

```bash
docker build -t flutter-app .
```

Run the container:

```bash
docker run -p 8080:8080 flutter-app
```

---

## Using Docker Compose

Start application:

```bash
docker compose up --build
```

Run in detached mode:

```bash
docker compose up -d
```

Stop application:

```bash
docker compose down
```
