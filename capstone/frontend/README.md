# Frontend - Flutter Application

## Overview

The frontend is built using Flutter and provides the user interface for the application. It communicates with backend services through REST APIs and delivers a responsive cross-platform experience.

## Technologies

* Flutter
  
* Dart
  
* Material Design

## Features

* Responsive UI
  
* Form Validation
  
* State Management
  
* API Integration
  
* Local Data Persistence
  
* Cross-Platform Support


## Installation

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```

## Build

Android APK:

```bash
flutter build apk
```

Web Build:

```bash
flutter build web
```

## Docker

Build image:

```bash
docker build -t flutter-frontend .
```

Run container:

```bash
docker run -p 8080:8080 flutter-frontend
```
