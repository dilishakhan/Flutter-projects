# Flutter Web Application 

## Overview

This project is a Flutter Web application that demonstrates frontend development, API integration, form validation, and data management using a Mock API. The application is designed to simulate real-world client-server interactions during development without requiring a production backend.

## Features

* Responsive Flutter Web UI
  
* REST API Integration
  
* Form Validation
  
* Mock API using JSON Server
  
* Docker Containerization
  
* Docker Compose Support

## Application Workflow

- User interacts with the Flutter Web application.
   
- Flutter sends HTTP requests to the Mock API.
   
- JSON Server processes the requests.
   
- Data is returned to the Flutter application.
   
- UI updates dynamically based on API responses.
